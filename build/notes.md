## Live Transcribe 1.4.0

Lokale Sprachdiktier-App mit OpenAI-Unterstützung.

**Neu in dieser Version**
- **Hintergrundinformationen für die KI** (Einstellungen → OpenAI): Hinterlege einmal Name, Firma,
  Rolle, Signatur oder Schreibstil. Diese Infos fließen in **jede** OpenAI-Überarbeitung ein — so
  kommt z. B. „Schreibe eine Mail an Herrn Büttner" mit passender Anrede und Signatur heraus.

**Aus 1.3.x**
- Assistent-Modus (Standard): Anweisung sprechen → nur das fertige Ergebnis wird eingefügt.
- Sichtbarer OpenAI-Status im Overlay („mit OpenAI" bzw. Grund bei Fehlern).

**Aus 1.2.x**
- Automatisches Einfügen ins fokussierte Textfeld (sonst Zwischenablage), lesbare dunkle Dropdowns.

> Hinweis: Für die OpenAI-Überarbeitung wird ein gültiger OpenAI-Key **mit Guthaben** benötigt
> (Einstellungen → API-Key). Ohne Guthaben wird der Rohtext eingefügt und der Grund angezeigt.

> Hinweis: Beim ersten Start kann Windows SmartScreen eine Warnung anzeigen, da das
> Installationsprogramm (noch) nicht signiert ist. „Weitere Informationen" → „Trotzdem ausführen".
